# AutoWorkshop Management System

## Project Name
**AutoWorkshop Management System** — консольное приложение для управления сетью автомастерских.

## Description
Система предназначена для автоматизации работы городской службы автомастерских. Приложение позволяет вести учет мастерских, мастеров, автомобилей и выполненных ремонтов. Реализована ролевая модель доступа: администратор имеет полный доступ к управлению, мастер — только к своим ремонтам. Система поддерживает формирование отчетов по мастерским и мастерам, расчет выручки, а также автоматическое обновление статистики через триггеры базы данных.

## Installation

### Требования
- ОС: Linux (Ubuntu/Debian) или macOS
- Компилятор: GCC
- Библиотеки: SQLite3, OpenSSL
- Утилиты: make, lcov (для отчета о покрытии)

### Установка зависимостей (Ubuntu/Debian)
```bash
sudo apt update
sudo apt install gcc make sqlite3 libsqlite3-dev libssl-dev lcov
````

### Клонирование репозитория с кодом
```bash
git clone https://github.com/your-username/autoworkshop-app.git
cd autoworkshop-app
```

### Сборка проекта
```bash
make clean
make all
```

### Инициализация базы данных
```bash
make init-db
```

## Usage

### Запуск приложения
```bash
./bin/autoworkshop
```

### Вход в систему
- **Администратор:** логин `admin`, пароль `password`
- **Мастер:** логин `ivanov`, пароль `password` (и другие мастера из БД)

### Меню администратора
- Управление мастерскими (CRUD)
- Управление мастерами (CRUD)
- Управление автомобилями (CRUD)
- Отчеты:
  - по мастерской за период
  - по мастеру
  - по маркам автомобилей
  - общая выручка
  - самая активная мастерская

### Меню мастера
- Просмотр своих ремонтов
- Добавление нового ремонта
- Обновление статуса ремонта
- Поиск автомобиля по госномеру
